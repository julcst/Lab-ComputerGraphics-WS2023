\section*{\GetTranslation{report-abstract}}
\label{sec:abstract}

In this report we will present the approach of \textcite{belcour2018} as well as the improvements \textcite{weier2020,yamaguchi2019} made to it. The key insight is that GGX lobes projected to the base plane can be approximated using circular/elliptical distributions. Therefore, light transport can be approximated using a statistical framework. This allows to evaluate light transport in layered materials efficiently. The resulting implementation is capable to reproduce a variety of complex surfaces, even in real-time.
