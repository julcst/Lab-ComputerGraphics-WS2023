\section*{\GetTranslation{report-abstract}}
\label{sec:abstract}

In this report, we explore the work of \textcite{deliot2023} to simulate glittery appearances caused by sub-pixel surface details which we will model procedurally.
%
Therefore, we describe a fast and stable stochastic counting process, that smoothly blends between different levels of an anisotropy compensating surface discretization to achieve constant runtime.
%
This differentiates it from previous works, that suffer from blending artifacts and varying performance.
%
All in all, we achieve a visually convincing implementation well capable of real-time.
